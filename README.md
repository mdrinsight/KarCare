# Kar Care Service - Premium Deployment Package

## Deployment Instructions for GitHub Pages (₹0 Cost):
1. Create a free GitHub Account.
2. Create a new **Public Repository** named `kar-care-service`.
3. Upload all 7 files (5 HTML, 1 CSS, 1 JS) directly into the repository main root.
4. Go to **Settings** -> **Pages**.
5. Under **Build and deployment** > **Source**, select **Deploy from a branch**.
6. Set the branch to **main** and click **Save**.
7. Wait 60 seconds and your 2 Lakh INR-tier web app will be globally live!
